1. Prefer TypeScript over JavaScript
Use .ts and .tsx instead of .js and .jsx.
It reduces bugs and improves developer experience.

2. Consider React Query (TanStack Query)
Instead of manually caching data in Zustand, use React Query for:
Server state
Automatic refetching
Caching
Optimistic updates

3. Refresh tokens
JWT alone can be limiting.
Implement short-lived access tokens with refresh token rotation.

4. Logging and monitoring
Add centralized logging (e.g., Pino/Winston) and error tracking.

5. Testing
Include:
Backend: integration tests
Frontend: component tests and end-to-end tests

6. API documentation
Generate OpenAPI/Swagger documentation for easier maintenance.

7. Containerization
Add Docker and Docker Compose for consistent deployment environments.

8. CI/CD pipeline
Configure GitHub Actions (or similar) for linting, testing, and deployment.

9. feature-based (domain-driven) structure
